Команди модератора Ukraine GTA5

Вітаю, шановний Discord Moderator! У цьому розділі ви можете ознайомитися з командами, доступними для модераторів Discord.

Система команд модерації

Нижче наведено список усіх доступних команд для керування користувачами та модерації сервера.

Одиниці часу для покарань

m
хвилини
h
години
d
дні

Приклади використання:

!mute @користувач 30m P.2
Видати мут на 30 хвилин за пункт правил 2
!mute @користувач 2h P.1
Видати мут на 2 години за пункт правил 1
!ban @користувач 15d P.11
Видати бан на 15 днів за пункт правил 11

Приклади команд

!mute ID 30m P.2
Мут на 30 хвилин за порушення пункту 2 правил
!mute ID 2h P.1
Мут на 2 години за порушення пункту 1 правил
!ban ID 15d P.11
Бан на 15 днів за порушення пункту 11 правил
!warn ID P.3
Попередження за порушення пункту 3 правил
!kick ID P.5
Кік за порушення пункту 5 правил

Зняття покарань

Зняття муту

!unmute ID

Зняти мут з користувача за вказаним ID

Зняття бана

!unban ID

Зняти бан з користувача за вказаним ID

Очищення попереджень

!clearwarns ID

Очистити всі попередження користувача

Додаткові команди

!clear 10
Видалити 10 останніх повідомлень у каналі
!userinfo ID
Переглянути інформацію про користувача
!serverinfo
Переглянути інформацію про сервер
!modstats
Переглянути статистику модерації
